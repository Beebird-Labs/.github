# BeeBird Labs

BeeBird Labs is a small organization focused on JDM electronics and related tooling.

## Public now

* [Toyota BEAN](https://github.com/Beebird-Labs/toyota-bean) — Python tools and notes for working with Toyota BEAN messages
* [ESP8266 BEAN Reader](https://github.com/Beebird-Labs/toyota-bean)](https://github.com/Beebird-Labs/ESP8266-BEAN-Reader) — An ESP8266-based BEAN reader that sends specific messages out via ESPNow

More tools, documentation, and experiments are in progress.
